## 프로젝트 소개

**QuickService**는 빠르고 효율적인 서비스 제공을 목표로 하는 웹 애플리케이션입니다. 

### 주요 기능

- 주문 생성
- 카카오 맵 API
- 채팅방(고객, 라이더 1:1 채팅방)
- 목록 조회/상세 조회
- 회원가입/고객 및 라이더 로그인 (아이디/비밀번호 찾기)
- 파일 첨부

### 사용된 기술 스택

- **프론트엔드**:
  - HTML5
  - CSS3
  - JavaScript
- **백엔드**:
  - Spring Framework
- **데이터베이스**:
  - Oracle Database 21c Express Edition
- **개발 환경**:
  - Java(TM) 플랫폼: 11.0.21, 0.2
  - Oracle IDE: 23.1.1.345.2114
  - 버전 지정 지원: 23.1.1.345.2114
